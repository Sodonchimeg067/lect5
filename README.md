## Өгөгдлийн сангийн удирдлагын бичиг баримт болон нууц үгийн бодлогыг хэрэгжүүлэх

### Асуулт
```
Plain text байдлаар нууц үг хадгалсан тохиолдолд үүсэх аюулгүй байдлын эрсдэлийг кейс жишээгээр тайлбарлана уу.
```

### Хариулт

Нууц үгийг ямар нэгэн хамгаалалтгүйгээр шууд хадгалахыг plain text хадгалалт гэнэ. Нууф үгийг plain text хэлбэрээр хадгалах нь мэдээллийн аюулгүй байдлын хамгийн том алдаануудын нэг. Үүнд дараах эрсдэлүүд үүснэ:
* **Мэдээлэл алдагдах:** Өгөгдлийн сан руу нэвтрэх эрх авсан хакер бүх хэрэглэгчийн нууц үгийг шууд харах боломжтой.
* **Давтан халдлага:** Хүмүүс ихэвчлэн өөр өөр платформ дээр ижил нууц үг ашигладаг учир нэг системээс алдагдсан нууц үгээр бусад хаяг руу нь нэвтрэх боломжтой болно. 

### Жишээ
2012 онд "LinkedIn" сүлжээнээс 6,5 сая хэрэглэгчийн нууц үг алдагдсан. Тухайн үед нууц үгийг salt ашиглалгүйгээр зөвхөн хадгалж байсан тул хакердаж чадсан байсан хэрэв plain text байсан бол ямар ч хүчин чармайлтгүйгээр шууд ашиглагдах байсан гэж үздэг.
